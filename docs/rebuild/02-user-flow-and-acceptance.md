# 使用者流程與驗收標準

## 訪客流程

1. 進首頁 -> 看到文章列表
2. 點文章 -> 進入詳情頁
3. 閱讀內容（含 code block）-> 可分享
4. 查看留言與留言排序/搜尋

### 驗收標準

- 首頁至少顯示 10 篇文章卡片
- 每篇文章卡片含：標題、摘要、作者、日期、留言數
- 詳情頁 code block 顯示一致，桌機/手機不跑版
- 無登入時留言輸入框顯示「請先登入」

## 會員流程

1. 社群登入（Line/Facebook/Google）
2. 在文章頁留言
3. 可操作按讚/取消讚

### 驗收標準

- 登入後 2 秒內可看到會員狀態
- 送出留言成功後，清單可立即看到新留言
- 同裝置重整後，按讚狀態維持一致

## 管理者流程

1. 建立文章 -> 預覽 -> 發佈
2. 更新文章 -> 確認 SEO 欄位
3. 管理留言（隱藏/刪除）

### 驗收標準

- 發佈後文章可於列表頁與 sitemap 出現
- 更新文章後 `lastmod` 可同步更新
- 隱藏留言後，前台留言數量正確下降

## SEO 驗收

- `/article?page=2` 可直接存取（非 JS-only）
- `sitemap.xml` 含首頁、列表、公開文章
- 所有舊文章 URL 轉址到新 canonical URL
